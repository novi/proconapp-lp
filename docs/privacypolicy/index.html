プライバシーポリシー

運営者は運営者が提供する「高専プロコンApp(各プラットフォーム上のApp, *.proconapp.comのWebサービスを含む)」(以下、本サービス)の利用者(以下、ユーザー)の 個人情報を、以下示すプライバシーポリシーに基づいて取得・使用します。 本サービスを利用するにはプライバシーポリシー(本Webページ)と利用規約(App Store)および利用規約(Microsoft Store)への同意が必要です。プライバシーポリシーおよび利用規約の同意無しに本サービスを利用することはできません。

個人情報の定義

個人情報とはユーザー個人を識別可能な情報をいい、本サービス利用時にユーザーが登録した、メールアドレス・パスワード等ユーザーに関する情報を含みます。

個人情報の利用目的・範囲

本サービスはユーザーの個人情報を次の目的および範囲内で使用します。

  • 本サービス提供・管理・運営・機能および内容向上のため
  • ユーザーが本サービスを利用するにあたり、ユーザーに必要な連絡を取るため
  • 本サービス上で個人を特定できない範囲の統計資料の作成・利用・公開のため

個人情報の第三者への提供

運営者は個人情報を次の場合を除き、ユーザーの同意無く、第三者へ提供することはありません。

  • 法令に基づく場合
  • ユーザーの同意を得た場合
  • 国・政府の機関、公共団体またはそれに準ずる者から提示を求められた場合
  • ユーザーまたは第三者の生命・身体・財産の保護のために必要があり、ユーザーからの同意を得るのが困難な場合

アカウントの削除

ユーザーの意思で、ユーザー自身が本サービスを端末上から削除することにより、ユーザーが保存した個人情報の一切を削除することができます。

また、サービス内で登録したメールアドレス等の個人情報に関しては、運営者まで連絡することによって削除することができます。

免責事項

本サービスの範囲外におけるWebサイト等の個人情報についてはプライバシーポリシーの範囲外であり、運営者が責任を負うものではありません。

運営者のセキュリティ保護・対策にも関わらず、クラッカー・ハッカーなどによる攻撃により、ユーザーおよび第三者に損害が生じた場合、運営者は責任を負いません。

改定

運営者はプライバシーポリシーをユーザーの同意を得ることなく変更可能で、変更後のプライバシーポリシーは公開した日から実効し、ユーザーが 本サービスを利用することにより、ユーザーは変更後のプライバシーポリシーに同意したものとします。
